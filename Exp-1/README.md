# Exp-1: React Fundamental Projects

A collection of essential React applications exploring core concepts like state management, theme switching, and SPA structures.

## Projects Included

1. **Theme Change (/Themechange)**
   - Implements dynamic theme switching logic.
   - Demonstrates React state-driven styling.

2. **Simple Form SPA (/simple-form-spa)**
   - Focused on form handling, data binding, and user input validation.
   - Shows best practices for managing complex form states.

3. **Todo List (/todo-list)**
   - A classic task management application.
   - Features include task addition, deletion, and persistent state using LocalStorage.

4. **Vertical Counter (/vertical-counter)**
   - A UI-focused component demonstrating vertical stacking and animation.
   - Useful for understanding component-level state and layout.
